# 🧮 C++ Advanced Calculator

## 📌 Overview
This is a simple console-based calculator built in C++. It performs basic and advanced mathematical operations using a menu-driven interface. The project is designed to improve understanding of loops, conditional statements, switch cases, and mathematical functions in C++.

---

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Power (^ using `pow()` function)
- Continuous calculations using loop
- User-friendly menu system
- Input validation for division/modulus by zero

---

## 🧠 Concepts Used
- Input/output in C++
- Conditional statements (if-else)
- Switch case
- Loops (do-while)
- Functions from `<cmath>` library
- Type casting (for modulus operation)

---

## 💻 How It Works
1. User enters the first number  
2. Selects an operation from the menu  
3. Enters the second number  
4. Program performs the calculation  
5. Result is displayed  
6. User can choose to continue or exit  

---

## 📋 Example Usage

```
Enter first number: 5

Available Operations:
+  Addition
-  Subtraction
*  Multiplication
/  Division
%  Modulus
^  Power

Choose an operator: *

Enter second number: 4

Result: 5 * 4 = 20
```

---

## 🔁 Loop Feature
After each calculation, the program asks:

- Press `y` → perform another calculation  
- Press `n` → exit program  

---

## ⚠️ Error Handling
- Prevents division by zero  
- Prevents modulus by zero  
- Handles invalid operator input
